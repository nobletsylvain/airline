# Governance & Ownership — Detailed Specification

**Version:** 0.1  
**Date:** January 2026  
**Companion to:** Game Design Document v0.7, Financial Model Spec v0.2

---

## Overview

This document specifies how ownership, investors, board governance, and stakeholder relations work in Airliner — the structures that sit above day-to-day operations and shape strategic direction.

**Design Philosophy:** Governance creates tension. You don't run the airline alone. Investors want returns, boards demand accountability, stakeholders have expectations. Success means balancing these interests while pursuing your vision.

**Core Loop:** Set strategy → Execute → Report results → Manage expectations → Navigate pressure → Repeat

## 1. Ownership Structure

### 1.1 Owner Types

| Type | Characteristics | Typical Stake | Expectations |
|------|-----------------|---------------|--------------|
| **Founder** | You, the player | 10-100% | Vision realization |
| **Private investor** | VC, PE, family office | 10-49% | ROI, exit timeline |
| **Strategic investor** | Other airline, supplier | 5-25% | Synergies, influence |
| **Government** | National/regional | 10-51% | Jobs, routes, prestige |
| **Public shareholders** | Post-IPO | 20-80% | Returns, dividends |
| **Employee** | ESOP | 5-20% | Job security, voice |

### 2.3 Investor Sentiment

| Level | Description | Consequences |
|-------|-------------|--------------|
| **Pleased** | Exceeding expectations | Support for plans, follow-on |
| **Satisfied** | Meeting expectations | Normal operations |
| **Watching** | Concerns emerging | More scrutiny, questions |
| **Concerned** | Missing targets | Pressure, restrictions |
| **Unhappy** | Significant underperformance | Board action, exit pressure |
| **Hostile** | Loss of confidence | Removal attempts, fire sale |

## 3. Board of Directors

### 3.1 Board Composition

| Role | Typical Count | Purpose |
|------|---------------|---------|

### 3.3 Board Decisions

| Decision Type | Board Role | Your Autonomy |
|---------------|------------|---------------|
| **Day-to-day ops** | Not involved | Full |
| **Route changes** | Informed | Full (unless restricted) |
| **Aircraft orders >$50M** | Approval required | Proposal + advocacy |
| **Senior hires** | Approval for C-suite | Recommendation |
| **M&A** | Approval required | Proposal |
| **Strategy changes** | Approval required | Proposal |
| **Financing >$20M** | Approval required | Proposal |
| **CEO compensation** | Board decides | No vote |
| **CEO removal** | Board decides | Defense only |

### 3.4 Board Confidence

| Level | Effect |
|-------|--------|
| **Strong** | Rubber-stamp approvals, bonus awarded |
| **Good** | Normal oversight, trust extended |
| **Mixed** | Scrutiny increases, conditions attached |
| **Low** | Restrictions imposed, milestones required |
| **Critical** | CEO position at risk, emergency measures |

## 5. Corporate Actions

### 5.1 IPO Process

| Stage | Duration | Requirements |
|-------|----------|--------------|
| **Preparation** | 6-12 months | Audited financials, governance |
| **Filing** | 2-3 months | Prospectus, regulatory review |
| **Roadshow** | 2-4 weeks | Investor presentations |
| **Pricing** | 1 day | Set offer price |
| **Trading** | Ongoing | Public company obligations |

**IPO readiness checklist:**
- 3 years audited financials
- Independent board majority
- Internal controls
- Management depth
- Growth story

### 5.2 M&A Activity

| Action | As Acquirer | As Target |
|--------|-------------|-----------|
| **Strategic merger** | Expand network | Exit for shareholders |
| **Acquisition** | Eliminate competitor | Premium to shareholders |
| **Sale** | N/A | Strategic exit |
| **Hostile takeover** | Rare in airlines | Defense needed |

### 5.3 Restructuring

| Type | Trigger | Outcome |
|------|---------|---------|
| **Recapitalization** | Debt too high | Debt-to-equity conversion |
| **Chapter 11** (US) | Cash crisis | Court protection, restructure |
| **Administration** (EU) | Insolvency | Restructure or liquidate |
| **Government rescue** | National interest | State support + conditions |

## 6. Data Model Integration

### 6.1 Entities

**Ownership** — Shareholding records

| Field | Type | Purpose |
|-------|------|---------|
| `id` | UUID | Primary key |
| `airline_id` | FK | The airline |
| `owner_type` | enum | FOUNDER / INVESTOR / PUBLIC / etc. |
| `owner_id` | UUID? | FK to Person or Investor |
| `owner_name` | string | Display name |
| `stake_percent` | float | Ownership % |
| `voting_percent` | float | Voting % (may differ) |
| `acquired_date` | date | When acquired |
| `acquisition_type` | enum | FOUNDING / INVESTMENT / IPO / etc. |
| `board_seats` | int | Board representation |
| `terms` | json | Investment terms |

**Investor** — External capital providers

| Field | Type | Purpose |
|-------|------|---------|
| `id` | UUID | Primary key |
| `name` | string | Investor name |
| `type` | enum | VC / PE / FAMILY_OFFICE / etc. |
| `contact_person_id` | FK → Person | Key contact |
| `investment_horizon` | int | Expected years |
| `return_target` | float | IRR or multiple target |
| `sentiment` | enum | PLEASED through HOSTILE |
| `concerns` | string[] | Current issues |
| `portfolio` | json | Other investments |

**Board** — Board composition

| Field | Type | Purpose |
|-------|------|---------|
| `id` | UUID | Primary key |
| `airline_id` | FK | Which airline |
| `overall_confidence` | enum | STRONG through CRITICAL |
| `meeting_frequency` | enum | MONTHLY / QUARTERLY |
| `next_meeting` | date | Scheduled meeting |
| `chair_id` | FK → BoardMember | Who chairs |

**BoardMember** — Individual directors

| Field | Type | Purpose |
|-------|------|---------|
| `id` | UUID | Primary key |
| `board_id` | FK | Which board |
| `person_id` | FK → Person | The person |
| `role` | enum | CHAIR / CEO / INVESTOR_REP / INDEPENDENT |
| `represents_id` | FK? → Investor | If investor rep |
| `joined_date` | date | Appointment date |
| `confidence_in_ceo` | enum | STRONG through CRITICAL |
| `focus_areas` | string[] | Expertise/interests |
| `concerns` | string[] | Current issues |

**BoardMeeting** — Meeting records

| Field | Type | Purpose |
|-------|------|---------|
| `id` | UUID | Primary key |
| `board_id` | FK | Which board |
| `date` | date | Meeting date |
| `type` | enum | REGULAR / EMERGENCY / SPECIAL |
| `agenda` | json | Topics discussed |
| `decisions` | json | Outcomes |
| `confidence_change` | int | Net change |

**Stakeholder** — Non-owner stakeholder groups

| Field | Type | Purpose |
|-------|------|---------|
| `id` | UUID | Primary key |
| `airline_id` | FK | Which airline |
| `type` | enum | SHAREHOLDERS / CREDITORS / EMPLOYEES / etc. |
| `sentiment` | enum | PLEASED through HOSTILE |
| `trend` | enum | IMPROVING / STABLE / DECLINING |
| `top_concern` | string | Primary issue |
